녹화된 영상, 이미지 등에서 자동으로 얼굴을 인식하여 필터를 입히거나 스티커를 붙이는 등의 효과를 줄 수 있습니다
이는 다양한 방면에서 굉장히 유용하게 사용될 수 있는 기술입니다.
